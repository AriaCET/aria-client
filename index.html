Welcome to the ARIA Project.

An public addressing system based on the Asterisk VoIP network. Asterisk can be made to run as paging system, but this feature is not given much importance. The steps are very crude and the resulting system is painful for a user to use. We are trying to streamline this process and is trying to provide a easy one-click aproach. It uses the Asterisk Telephony server at the back and uses SIP/RTP protocols for data transfer.

Setting up the system.


$ git clone git://github.com/abilng/aria_client.git
$ cd src
$ ./ariasetup.py #sets up the password, Registration details & Speaker details.
$ ./main.py

Dependencies

  • python 2
  • python-pjsua
  • PySide or PyQt
